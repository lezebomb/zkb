# 比赛当天 Runbook

## 开赛前 30 分钟

1. 开机并连接稳定网络，优先有线。
2. 检查 Python 环境和依赖是否完整。
3. 启动服务：`uvicorn contest_agent.app:app --host 0.0.0.0 --port 8080`
4. 测 `/health`：`curl http://127.0.0.1:8080/health`
5. 跑三类 `/infer` 烟测：`bash scripts/smoke_test.sh`
6. 检查 `logs/app.log`，确认没有异常堆栈和超时告警。

## 登录平台

1. 使用队长账号登录。
2. 进入算法网关配置，填写或确认 Agent Base URL。
3. 地址示例：`http://127.0.0.1:8080`
4. 点击检测连接，确保平台可访问本地服务。

## 开始计时

1. 进入「开赛与计时」页面。
2. 确认所有前置检查绿色通过。
3. 必须在 `2026-07-04 13:30` 前点击开始计时。

## 视觉评测

1. 先跑最稳的槽位，优先保证接口正确和 `detect` 可用。
2. 每个槽位机会有限，失败后先查日志，不要盲目重复。
3. 重点检查：
   - `request_id` 是否原样回显
   - `detect` 是否输出像素 `cx/cy`
   - `ocr` 是否返回 `result.text`
   - 是否出现非 JSON 或超时

## 在线答题

1. 先做把握高的单选。
2. 简答题按 `docs/short_answer_templates.md` 快速改写。
3. 按“概念定义 -> 工业场景 -> 工程落地 -> 风险与优化”组织答案。

## 整场提交

1. 确认「在线答题」和「视觉评测」都已完成。
2. 队长点击「提交并结束」。

## 赛后 3 小时内

1. 运行 `bash scripts/package_submission.sh`
2. 检查压缩包大小不超过 `2GB`
3. 上传算法文件到平台附件区
4. 确认包含运行说明、依赖、脚本和必要权重，保证可复现

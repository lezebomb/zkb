# 在线答题复习提纲

## 1. 机器视觉与数字图像基础

- 像素与分辨率：像素是采样单位，分辨率决定细节上限，也影响算力和存储。
- 灰度、RGB、HSV：灰度适合结构分析，RGB适合原始成像，HSV便于颜色分离和阈值设计。
- 图像增强：直方图均衡、Gamma、对比度拉伸常用于提升弱对比工业画面可读性。
- 滤波：均值滤波抑制随机噪声，中值滤波抗椒盐噪声，高斯滤波兼顾平滑与边缘保留。
- 几何校正：解决透视、旋转、尺度变化；常与模板定位或标定板配合。
- 相机标定：求内参、外参和畸变参数，为测量和三维重建打基础。
- 畸变：桶形、枕形会影响测量与定位精度，工业场景通常要先矫正。
- 手眼标定：建立相机坐标系与机械臂坐标系关系，是抓取和引导定位关键环节。

## 2. 传统图像处理与特征

- 边缘与角点：边缘适合轮廓提取，角点适合配准、定位和跟踪。
- 霍夫变换：适合检测直线、圆等规则结构，工业仪表和管道场景常见。
- 轮廓分析：可用于面积、周长、长宽比、圆度等几何特征筛选。
- 模板匹配：适合姿态变化小、背景相对稳定的重复工件定位。
- 形态学：腐蚀、膨胀、开闭运算常用于去噪、补孔和连通区域整形。
- 阈值分割：全局阈值简单高效，自适应阈值适合光照不均。
- 纹理与频域：纹理特征常用于表面缺陷，频域有助于周期噪声抑制和结构分析。

## 3. 深度学习与典型视觉任务

- CNN核心：卷积提局部特征，池化降采样，BN稳定训练，激活函数增强非线性表达。
- 分类：输出整图类别，适合场景识别、工件状态判断。
- 检测：同时定位与分类目标，可分两阶段和单阶段。
- 两阶段检测：精度通常较高，但速度和部署成本偏高。
- 单阶段检测：速度快，适合在线评测和嵌入式部署。
- 分割：
  - 语义分割：像素级类别，不区分个体。
  - 实例分割：区分同类不同实例。
  - 全景分割：语义分割和实例分割结合。

## 4. 三维视觉、SLAM 与传感器

- 双目视觉：基于视差恢复深度，成本适中，依赖纹理和标定质量。
- 结构光：主动投射编码光，近距离精度高，但易受环境光影响。
- 线激光：适合轮廓测量、焊缝跟踪和尺寸检测。
- 点云：表达三维几何，可做配准、分割、姿态估计。
- 多视图重建：通过多个视角恢复三维结构，适合复杂场景建模。
- 视觉里程计与SLAM：前者重估位姿，后者同时建图与定位。
- 传感器选型：看量程、精度、刷新率、抗反光能力、现场集成难度。

## 5. 工业场景与工程落地

- 缺陷检测：核心难点是样本不平衡、缺陷稀少、现场域偏移。
- 测量：关键是标定、稳定成像和误差传递控制。
- 定位引导：强调重复精度、实时性、坐标系统一和末端执行器误差补偿。
- 读码与OCR：要关注反光、污渍、弯曲表面、字符规范化。
- 光源与镜头：先解决成像，再谈算法；背光、同轴光、低畸变镜头都很重要。
- 嵌入式部署：关注模型体积、显存、功耗、热设计和推理延迟。
- 优化手段：量化、剪枝、蒸馏、TensorRT/ONNX Runtime 加速、批处理与流水线优化。

## 6. 拓展与综合

- Vision Transformer：擅长全局建模，但通常更依赖数据规模和算力。
- 自监督学习：降低标签依赖，适合工业场景标注不足。
- 小样本学习：解决新类别样本少的问题，但泛化要结合场景验证。
- 域适应：解决实验室到产线、白天到夜间、相机替换等分布偏移。
- 多模态：把图像、文本、时序信号、点云等结合，提高系统鲁棒性。
- 开放集检测：面对未知类和异常样本时，重点是拒识与风险控制。
